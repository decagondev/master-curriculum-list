Code-Alongs

What is a Code-Along?

Code-Alongs are live experiences taught by our instructors designed to prepare you for concepts found in the sprint challenges. They're your opportunity to work on complex job-ready problems in a live and engaging environment.

These sessions are 50 minutes in length and are offered seven days a week in the morning, afternoon, and evening. Because Code-Alongs delve deeper into a core competency, you will need to come to class prepared to have the best experience.

Ideal Code-Along Preparation Checklist

  • Did you review the core competencies?
  • Did you watch the guided projects?
  • Did you finish your module projects?

Code-Along: Advancing with LeetCode, Problem-Solving Strategies

This Code-Along focuses on developing advanced problem-solving techniques for LeetCode challenges. You'll work through increasingly complex problems that build on your Python data structure knowledge and algorithmic thinking. These exercises will strengthen your technical interview skills and help you tackle harder coding assessments with confidence.

4Sum (Hard Array Sum Problem) Find All Anagrams in a String Fraction to Recurring Decimal

How to Prepare

  • Review the technical preparation concepts from Modules 1-3
  • Practice with basic LeetCode problems
  • Complete your module projects before attending
  • Have a LeetCode account ready and Python environment set up

Code-Along: Machine Learning Application on Language Data

This Code-Along explores machine learning techniques applied to language data. You'll learn how to process text data, extract features, and build models for natural language processing tasks. These skills are essential for data scientists working with text-based applications such as sentiment analysis, topic modeling, and text classification.

Prepare for Success

The best Code-Along experiences happen when you are ready before coming to class. Your instructors created a starting point and a solution for each of your Code-Alongs to ensure you have what you need to succeed.

Make sure to review the relevant module materials before attending the code-along. This will help you get the most out of the session and be better prepared to tackle the challenges.

Welcome to DS Unit 3 Sprint 12

In this sprint, you will continue to improve upon your programming and problem-solving skills.

Tips on how to succeed in this sprint:

  • Watch all the lessons and guided projects
  • Attend ACS Live Classes
  • Complete all practice activities in each module
  • Take the practice and official General Code Assessment (GCA)
  • Build your resume using the BloomTech resume template

Module 4: Logistic Regression

Module Overview

In this module, you will transition from regression to classification with logistic regression. You'll implement train-validate-test splits, understand classification baselines, learn the intuition behind logistic regression, and work with scikit-learn pipelines to streamline your modeling workflow.

Learning Objectives

Objective 1: Use scikit-learn pipelines

Learn how to build efficient modeling workflows using scikit-learn pipelines.

  • Understanding the purpose and benefits of pipelines
  • Building preprocessing and modeling pipelines
  • Implementing feature scaling within pipelines
  • Using pipelines with cross-validation

Objective 2: What is a hyperparameter?

Learn about hyperparameters and how they differ from model parameters.

  • Understanding the distinction between model parameters and hyperparameters
  • Identifying common hyperparameters in machine learning models
  • Recognizing the impact of hyperparameters on model performance
  • Exploring approaches to hyperparameter selection

Objective 3: Implement train-validation-test split

Learn how to properly split your data into training, validation, and test sets for robust model evaluation.

  • Understanding the purpose of having three separate data splits
  • Implementing train-validate-test splits in scikit-learn
  • Using validation sets for hyperparameter tuning
  • Preserving test sets for final evaluation

Objective 4: Determine baseline for classification

Learn how to establish simple baseline models for classification problems.

  • Implementing majority class and stratified random baselines
  • Understanding classification metrics for baselines (accuracy, precision, recall)
  • Dealing with class imbalance in baselines
  • Using baselines to evaluate more complex classifiers

Objective 5: Fit and interpret Logistic Regression models

Learn how to implement logistic regression models using scikit-learn and interpret the results.

  • Implementing LogisticRegression in scikit-learn
  • Tuning regularization parameters (C parameter)
  • Interpreting logistic regression coefficients
  • Understanding probability outputs vs. class predictions

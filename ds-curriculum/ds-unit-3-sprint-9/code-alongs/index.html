About Code-Alongs

Code-Alongs are live experiences taught by our expert instructors designed to prepare you for concepts found in the sprint challenges. Code-Alongs are your opportunity to work on complex job-ready problems in a live and engaging environment.

Prepare For Success

The best Code-Along experiences happen when you are ready before coming to class. Your instructors created a starting point and a solution for each of your Code-Alongs to ensure you have what you need to succeed.

Ideal Code-Along Preparation Checklist:

  • Did you review the core competencies before coming to class?
  • Did you watch the guided projects before coming to class?
  • Did you take the checks for understanding before coming to class?
  • Did you finish your module projects before coming to class?

Code-Along 1: Python Packages and Virtual Environments

Overview

In this Code-Along, you'll learn how to create Python packages and work with virtual environments. This is essential knowledge for developing modular, maintainable Python code and for managing project dependencies effectively.

What You'll Build

You'll create a Python package from scratch, set up a virtual environment, and learn how to manage dependencies with pipenv. You'll also learn how to structure your code in a way that makes it reusable and maintainable.

Preparation

Before attending this Code-Along, it's recommended that you:

  • Complete Module 1: Python Modules, Packages, and Environments
  • Review the concept of Python namespaces and imports
  • Have Python and pipenv installed on your computer

Code-Along 2: Python Classes and Unit Tests

Overview

In this Code-Along, you'll learn how to create and use Python classes effectively, and how to write unit tests to ensure your code works as expected. These skills are fundamental for software engineering and will help you build more robust and maintainable code.

What You'll Build

You'll create Python classes that model real-world entities, and write unit tests to verify their functionality. You'll also learn about software testing principles and how to ensure your code works as expected.

Preparation

Before attending this Code-Along, it's recommended that you:

  • Complete Module 2: OOP, Code Style, and Reviews
  • Have a good understanding of object-oriented programming principles
  • Review the basics of software testing

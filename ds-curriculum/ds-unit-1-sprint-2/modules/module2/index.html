Module 2: Hypothesis Testing (chi-square tests)

We'll continue with the hypothesis testing but introduce a different way to quantify the significance of our results: the chi-square test. Next, we'll cover when to use a chi-square test and some of the typical applications. Finally, to expand on the previous module, we'll look at how to determine the p-value using a chi-square test and how to use it to interpret our null hypothesis.

Learning Objectives

By the end of this module, you should be able to:

  • Explain the purpose of a chi-square test and identify applications
  • Set up a chi-square test for independence on two categorical variables
  • Use a chi-square test p-value to draw the correct conclusion about the null and alternative hypothesis

Module 4: Linear Correlation and Regression

In this final module of the sprint, we'll explore the relationship between variables through linear correlation and regression. Building on our statistical foundation, we'll learn to quantify relationships between variables and build our first predictive models using linear regression equations.

Learning Objectives

By the end of this module, you should be able to:

  • Distinguish between independent and dependent variables
  • Distinguish between linear and non-linear relationships
  • Calculate the linear correlation using the scipy.stats library
  • Calculate and interpret the slope and intercept
  • Calculate a prediction and a residual

Key Concepts

  • Independent vs. dependent variables (features vs. target)
  • Creating and interpreting scatterplots
  • Pearson correlation coefficient
  • Positive, negative, and no correlation
  • Linear regression models
  • Interpreting slope and intercept
  • Making predictions with linear models
  • Calculating and understanding residuals

Module Project

Project Tasks

In this module's project, you will apply your knowledge of linear correlation and regression to real-world data. You will:

  • Identify dependent and independent variables in a dataset
  • Create scatterplots to visualize relationships between variables
  • Calculate correlation coefficients and interpret their meaning
  • Fit a linear regression model to your data
  • Interpret the slope and intercept parameters
  • Make predictions using your regression model
  • Calculate residuals to evaluate model fit

Complete all tasks in the Jupyter notebook DS_124_Simple_Linear_Regression_Assignment_AG.ipynb provided in the GitHub repository. Note that the video solution references a different notebook but the material is the same.

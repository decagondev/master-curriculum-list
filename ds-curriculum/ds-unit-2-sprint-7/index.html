Welcome to DS Unit 2 Sprint 7

In this sprint, you'll learn about applied modeling with a focus on defining machine learning problems, wrangling datasets, using permutation and boosting techniques, and interpreting your models. You'll work with various tools and techniques that data scientists use to explain and interpret their machine learning models.

Throughout the sprint, you'll apply your knowledge to real-world datasets, practice feature engineering, and learn how to handle imbalanced data. By the end of the sprint, you'll be able to build and interpret sophisticated machine learning models that go beyond simple predictions.

Sprint Overview

Module 1: Define ML Problems

Learn how to properly define machine learning problems, choose appropriate targets, understand target distributions, and select evaluation metrics.

Module 2: Wrangle ML Datasets

Explore techniques for data exploration, cleaning, and joining relational data in preparation for machine learning tasks.

Module 3: Permutation and Boosting

Understand feature importance through permutation techniques and implement gradient boosting with XGBoost for improved model performance.

Module 4: Model Interpretation

Learn to visualize and interpret model results with partial dependence plots and SHAP (SHapley Additive exPlanations) values.

How to Pass This Sprint

To successfully complete this sprint, you should:

  • Complete all module guided projects
  • Attend at least one code-along session
  • Pass all module checks for understanding
  • Complete and pass the sprint challenge

Focus on understanding the entire machine learning workflow from problem definition to model interpretation. The concepts in this sprint build upon each other, so make sure you have a solid understanding of each module before moving on.

Sprint Challenge - Web Unit 2 Sprint 7

Project Overview

In this sprint challenge, you will build a React application that demonstrates your understanding of React Router, Form Management, and Testing. You will create a multi-page application with form handling and comprehensive test coverage.

Project Requirements

React Router Requirements

  • Implement client-side routing using React Router
  • Create at least 3 different routes
  • Use URL parameters where appropriate
  • Implement programmatic navigation

Form Management Requirements

  • Create controlled form components
  • Implement form validation
  • Handle form submission
  • Display validation errors

Testing Requirements

  • Write unit tests for components
  • Write integration tests for form submission
  • Test routing functionality
  • Achieve at least 80% test coverage

Project Setup

# Clone the repository
git clone https://github.com/your-org/sprint-challenge-7.git

# Install dependencies
npm install

# Start the development server
npm start

# Run tests
npm test

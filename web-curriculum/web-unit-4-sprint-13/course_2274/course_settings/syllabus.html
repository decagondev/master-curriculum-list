Unit 4 Web Development

Unit Description:

Unit 4 will introduce you to the HTTP server! You will build a REST API to provide clients with data represented in JSON format. You will store information in a SQL database, and learn how to extract useful insights from the database tables using SQL queries. You will design your own database structure, and instantiate the physical database using the command line. And you will learn to write clean and maintainable server code using the ExpressJS library.

Submission Overview

Why Career Artifacts Matter

Your career artifacts serve as your professional introduction to potential employers. These materials showcase not only your technical skills but also your ability to present yourself professionally. They are essential tools in your job search toolkit and will often determine whether you get an interview opportunity.

Impact of Quality Artifacts:

  • Increased visibility to recruiters and hiring managers
  • Higher response rates from job applications
  • Better alignment with targeted roles
  • More effective communication of your technical abilities
  • Clearer representation of your professional brand

The Requirement

Submitting your career artifacts is a requirement for this unit. You will need to submit the following through our Google Form:

  • GitHub Profile

    Your professional GitHub profile showcasing your projects and contributions

    Key Elements: Professional README.md, pinned repositories, complete profile information, consistent coding style across projects
  • Resume

    Your resume following the required template format

    Key Elements: Clear technical skills section, quantified achievements, project descriptions with technologies used, education details, consistent formatting
  • LinkedIn Profile

    Your professional LinkedIn profile with updated experience and skills

    Key Elements: Professional photo, compelling summary, detailed experience section, listed skills with endorsements, education details, consistent with resume information

Submission Process

How to Submit

  1. Ensure all your career artifacts are complete and up to date
    Tip: Review each artifact against the quality checklist provided in class materials
  2. Visit our Career Artifacts Submission Form
    Note: The form link will be available in your Canvas assignment
  3. Fill in your personal information (name and email)
    Important: Use your BloomTech email address to ensure proper tracking
  4. Provide links to your:
    • GitHub profile URL
    • LinkedIn profile URL
    • Resume (hosted link or upload)
    Format: For GitHub and LinkedIn, provide the full URL (e.g., https://github.com/yourusername)
  5. Submit the form and await confirmation
    Timeline: You should receive an email confirmation within 24 hours

Feedback and Support

After submitting your career artifacts through the Google Form:

  • You will receive an email confirmation of your submission
  • Our career team will review your artifacts
    Review Criteria: Professional presentation, technical accuracy, alignment with industry standards, completeness
  • You will receive detailed feedback on each component
    Feedback Format: Specific suggestions for improvement, strengths, and areas for development
  • You may be asked to make improvements based on the feedback
    Iteration Process: You'll have an opportunity to implement changes and resubmit if necessary

Common Feedback Areas

GitHub

  • • Repository organization
  • • README clarity and completeness
  • • Code commenting practices
  • • Consistent commit messages

Resume

  • • Technical skills specificity
  • • Achievement quantification
  • • Project descriptions
  • • ATS optimization

LinkedIn

  • • Professional summary
  • • Experience details
  • • Skills section organization
  • • Profile completeness

Technical Example: Effective GitHub README

# Full Stack E-Commerce Application

## Project Overview
A complete e-commerce solution built with React, Node.js, Express, and MongoDB. Features include user authentication, product catalog, shopping cart, checkout process, and admin dashboard.

## Technologies Used
- **Frontend**: React, Redux, Material-UI, Styled Components
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Authentication**: JWT, bcrypt
- **Payment Processing**: Stripe API
- **Deployment**: Heroku, MongoDB Atlas

## Features
- Responsive design for desktop and mobile devices
- User authentication and authorization
- Product browsing with filtering and sorting
- Shopping cart persistence
- Secure checkout with Stripe
- Admin dashboard for inventory management
- Order tracking system

## Installation
```bash
# Clone the repository
git clone https://github.com/username/ecommerce-app.git

# Navigate to project directory
cd ecommerce-app

# Install dependencies for server
npm install

# Install dependencies for client
cd client
npm install

# Return to root directory
cd ..

# Create .env file and add your environment variables
# See .env.example for required variables

# Run development server
npm run dev
```

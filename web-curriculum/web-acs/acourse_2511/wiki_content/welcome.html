Module 1

 

Career Readiness

Description: This module covers how to submit your final career artifacts.

Module 2

ACS 5: Sprint 17 - Linked Lists, Stacks and Queues, Recursion

Description: This module covers how to search, insert into and delete from a linked list, the behavior of stacks and queues, and introduces recursion.

Module 3

ACS 6: Sprint 18 - Binary Trees and Searching

Description: This module covers the layout and structure of a binary search tree, how to apply recursion and queues to search, how to implement linear and binary search, and understand the complexities between the two.

Module 4

ACS 7: Sprint 19 - Sorting and Hash Tables

Description: This module covers basic O(n2) sorts, how to apply divide and conquer to merge sort/quick sort, and understand how to use hash tables.

Module 5

ACS 8: Sprint 20 - Graphs

Description: This module covers the importance of graphs and implementing a depth-first or breadth-first traversal using stacks and queues.

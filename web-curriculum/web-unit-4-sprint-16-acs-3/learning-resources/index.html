Important Information

Technical Requirements

Required:

  • Computer with 1.6 GHz Processor
  • 4 GB RAM
  • 120 GB Hard Drive
  • Web Cam
  • Microphone

Recommended:

  • 2 GHz Processor
  • 8 GB RAM
  • 256 GB SSD

Supported Operating Systems:

  1. macOS
  2. Windows 10 Pro
  3. Ubuntu

Chromebooks are not supported for this course.

Welcome to ACS 1!

This sprint is structured a little differently than what you have experienced so far.  What does this mean?

Well, this sprint is focused on two goals:

  1. Help you to prepare for and pass the General Code Assessment (GCA)
  2. Help you prepare for job search and get career ready.

How will this be done?

The first three modules in this sprint explore programming and problem-solving skills that will help you prepare for the GCA (goal 1 above).  The fourth module will guide you on how to get job search and career ready.

Every 4th sprint in the core program will be structured similar to this sprint and give you an opportunity to prepare for the GCA and build a career artifact.

Tips on how to succeed in this sprint:

  • Watch all the lessons and guided projects
  • Complete all practice activities in each module
  • Take the practice and official General Code Assessment (GCA) 
  • Start networking and outreach

Review Introduction to CodeSignal and General Code Assessment (GCA) and Unit Syllabus before you begin.

Module 1

Review of Basics

This module reviews basic coding knowledge and how to review, debug and submit a task in CodeSignal.

Module 2

Data Types and If-Then Statements

This module covers different data types, how to convert between them, and how to utilize Boolean programming logic and if-statements.

Module 3

Strings and Arrays

This module covers how to construct strings and for-loops as well as arrays and looping with while-loops.

Module 4

Career Readiness: Outreach & Networking + Resume

In this module, you'll get your first look at the job search, taking those crucial first steps toward career readiness. Networking and outreach will be introduced equipping you with essential tools for getting started. We'll also begin crafting your resume and explore roles you're already qualified for!

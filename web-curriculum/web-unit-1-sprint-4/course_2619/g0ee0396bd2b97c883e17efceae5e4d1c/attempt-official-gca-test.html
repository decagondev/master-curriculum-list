The GCA Requirement for this Sprint

It is time to attempt the official General Code Assessment (GCA) test!

Attempting this test is a requirement to pass this sprint.  You will attempt this test multiple times throughout the program.  This is so that you can determine areas that you may need to study more to earn a higher score.  Keep in mind that only your highest score is tracked in the BloomTech database, no matter how many times you take the test.

While in this sprint, you are not required to achieve a specific score, the target you should aim to achieve is 295 or higher.  

The score requirement becomes effective later in the program when you start the Applied Computer Science unit (more information will be shared when you start this unit). 

But there is no need to wait - you can study, practice, and prepare now to achieve a score of 295 or higher!

How to Prepare:

  • Watch the module core competency videos
  • Watch the guided project videos 
  • Attend live sessions
  • Complete the module practice activities
  • Practice using the CodeSignal Arcade

Once you have completed the above, follow the instructions below to take the official General Code Assessment test.

Instructions:

  1. You will take the official GCA and submit the test in CodeSignal.
  2. Login to CodeSignal* using the same email address you use to login to the portal.
  3. Take the official GCA HERE.
  4. Type your score in the Text Box below
  5. Check the Performance Page in the BloomTech Portal to ensure your score is registered.

You should have already created your CodeSignal account*, if you have not done so, please reference: What is CodeSignal and How to Create Your Account, CodeSignal User Interface Overview and Introduction to GCA.

*As a reminder, you must create your CodeSignal account using the SAME email address you use to login to the learning portal. Failure to do so will mean that your GCA test scores will not sync to the BloomTech database. 

What happens if the email I use to log in to CodeSignal to take the General Code Assessment (GCA) test is different than the email I use to log in to the Portal?

If the email logins don't match, your GCA test results will not be tracked in the BloomTech database, and you will have to retake the GCA using your matched login credentials.

Tip:

A score of 295 can be achieved by getting 100% on tests and passing any task. Task #1 is generally the easiest.  If you don't achieve the required score on this attempt, don't worry as you will get more chances to retake this assessment throughout the program.  Remember that there is a 2-week cool-off period before you can attempt the GCA again. 

Remember that the problems are intended not to have obvious solutions; hence you won't see a path to complete a problem immediately. It might take a while to understand the problem and develop a plan before you can begin coding. Don't rush into coding!

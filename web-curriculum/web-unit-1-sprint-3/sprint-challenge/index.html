Sprint Challenge

Overview

This sprint challenge will test your understanding of the concepts covered in Sprint 3, including:

  • Advanced JavaScript Objects and Classes
  • Advanced Functions and Closures
  • Real-world JavaScript Patterns
  • HTML & CSS III Concepts

Challenge Requirements

Complete the following tasks:

  • Implement object-oriented programming principles
  • Use advanced function concepts
  • Apply real-world JavaScript patterns
  • Create responsive layouts using modern CSS

Project Setup

  1. Fork and clone the sprint challenge repository
  2. Install dependencies using npm
  3. Start the development server
  4. Complete the required tasks
  5. Submit your work through CodeGrade

Submission Guidelines

  • Push your code to your fork of the repository
  • Submit the pull request URL to CodeGrade
  • Ensure all tests are passing
  • Include a README with setup instructions

Grading Rubric

  • Code Quality (30%)
  • Functionality (40%)
  • Documentation (15%)
  • Testing (15%)

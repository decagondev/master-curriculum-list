Sprint Challenge

In this sprint challenge, you'll apply everything you've learned about DOM manipulation, event handling, component creation, and working with APIs to build a complete web application.

Overview

You'll be building a data-driven web application that demonstrates your understanding of all the concepts covered in this sprint. The application will involve creating components, handling user interactions, and consuming data from an API.

Challenge Requirements

Core Requirements

  • Create reusable components using JavaScript classes
  • Implement event handling for user interactions
  • Fetch and display data from an API
  • Handle loading states and errors appropriately
  • Use proper DOM manipulation techniques

Stretch Requirements

  • Implement data filtering and sorting
  • Add animations for state changes
  • Implement error boundaries
  • Add unit tests for components

Project Setup

  1. Fork and clone the project repository
  2. Install dependencies using npm
  3. Start the development server
  4. Begin implementing the requirements

Submission Guidelines

  • Submit your code to GitHub
  • Include a README with setup instructions
  • Document any stretch features implemented
  • Provide screenshots of the working application

Grading Rubric

  • Core Requirements: 70%
  • Code Quality: 15%
  • Documentation: 10%
  • Stretch Features: 5%

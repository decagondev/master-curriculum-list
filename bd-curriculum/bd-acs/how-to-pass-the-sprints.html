How To Pass The Sprints

Understanding the requirements to successfully complete career and technical sprints.

How to Pass the Career Readiness Sprint

  • Submit final career artifacts in the Sprint Challenge section
  • Career artifacts will be reviewed and graded by a reviewer
  • Achieve 100% on all career artifacts

Career Artifacts Examples

  • Professional resume
  • LinkedIn profile
  • GitHub portfolio
  • Cover letter template

How to Pass the Technical Sprints

You must complete the Sprint Challenge Requirement:

  • Achieve a minimum score of 650 on the General Code Assessment (GCA) test
  • Complete the technical interview

GCA Preparation

  • Practice on LeetCode
  • Review Data Structures & Algorithms
  • Complete practice tests
  • Attend GCA prep sessions

Learning Objectives

Sprint Requirements Knowledge

  • Understand career sprint assessment criteria
  • Identify technical sprint requirements
  • Learn grading standards for career artifacts
  • Recognize GCA scoring thresholds

Success Strategies

  • Develop effective preparation techniques
  • Create high-quality career artifacts
  • Apply testing strategies for technical assessments
  • Implement feedback from reviewers

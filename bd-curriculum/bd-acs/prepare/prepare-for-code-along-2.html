What is a Code-Along?

Code-Alongs are live experiences designed to help you prepare for your upcoming Sprint Challenge. They are designed to be highly interactive, relevant to the content you are learning, and efficient in terms of time investment. Code-Alongs are not meant to be comprehensive or cover every possible topic in your Sprint. They are meant to provide you with a hands-on experience that will elevate your learning.

Common Data Structures Overview

During the Code-Along, we'll be working with several core data structures. Here's a brief overview:

Arrays

Arrays store ordered collections of values and are fundamental to many algorithms.

// Creating arrays const numbers = [1, 2, 3, 4, 5]; const mixed = [1, "two", { three: 3 }, [4]]; // Common operations numbers.push(6); // Add to end - O(1) numbers.pop(); // Remove from end - O(1) numbers.unshift(0); // Add to beginning - O(n) numbers.shift(); // Remove from beginning - O(n) numbers.splice(2, 1); // Remove at index - O(n) // Iteration for (let i = 0; i < numbers.length; i++) { console.log(numbers[i]); } // Functional methods const doubled = numbers.map(num => num * 2); const evens = numbers.filter(num => num % 2 === 0); const sum = numbers.reduce((acc, val) => acc + val, 0);

Objects

Objects are collections of key-value pairs, providing fast access to values through their keys.

// Creating objects const person = { name: "Alice", age: 28, isStudent: true }; // Adding properties person.location = "New York"; // Accessing properties console.log(person.name); // Dot notation console.log(person["age"]); // Bracket notation // Checking if property exists console.log("name" in person); // true // Iterating through properties for (const key in person) { console.log(`${key}: ${person[key]}`); } // Getting keys and values const keys = Object.keys(person); // ["name", "age", "isStudent", "location"] const values = Object.values(person); // ["Alice", 28, true, "New York"]

Sets

Sets store unique values and are useful for removing duplicates and checking membership.

// Creating a set const uniqueNumbers = new Set([1, 2, 3, 2, 1]); // {1, 2, 3} // Adding values uniqueNumbers.add(4); // {1, 2, 3, 4} uniqueNumbers.add(2); // No change, already exists // Checking for a value console.log(uniqueNumbers.has(3)); // true console.log(uniqueNumbers.has(5)); // false // Removing values uniqueNumbers.delete(1); // {2, 3, 4} // Size of set console.log(uniqueNumbers.size); // 3 // Iterating through values for (const num of uniqueNumbers) { console.log(num); }

Maps

Maps are key-value collections where keys can be of any type, unlike objects where keys are strings.

// Creating a map const userRoles = new Map(); // Adding entries userRoles.set("alice", "admin"); userRoles.set("bob", "editor"); userRoles.set("charlie", "viewer"); // Getting values console.log(userRoles.get("alice")); // "admin" console.log(userRoles.get("dana")); // undefined // Checking if key exists console.log(userRoles.has("bob")); // true // Size of map console.log(userRoles.size); // 3 // Deleting entries userRoles.delete("charlie"); // Iterating through entries for (const [user, role] of userRoles) { console.log(`${user} is a ${role}`); }

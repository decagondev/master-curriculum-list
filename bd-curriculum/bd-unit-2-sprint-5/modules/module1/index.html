Module 1: Functional Requirements

Module Overview

In this module, you'll learn how to create effective functional requirements for your Java applications. You'll understand how to identify and document requirements that are testable, measurable, and implementable.

Learning Objectives

  • Understand what makes a good functional requirement
  • Learn how to identify and document requirements
  • Understand the difference between functional and non-functional requirements
  • Learn how to create testable requirements
  • Practice writing user stories and acceptance criteria

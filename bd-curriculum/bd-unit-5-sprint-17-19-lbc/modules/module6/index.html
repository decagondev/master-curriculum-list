Introduction to Web Development

In this module, you'll learn the fundamentals of web development to create a user interface for your Learn and Be Curious project. Up until now, you've been building the backend of your application - the API endpoints and database. Now it's time to create a frontend that users can interact with.

Web development involves three core technologies:

  • HTML (HyperText Markup Language): The structure and content of web pages
  • CSS (Cascading Style Sheets): The presentation and styling of web pages
  • JavaScript: The behavior and interactivity of web pages

Together, these technologies allow you to create interactive web applications that connect to your backend services.

HTML Basics

Structure of an HTML Document

HTML documents have a basic structure that includes:

  • A doctype declaration
  • HTML, head, and body tags
  • Content within the body

Here's a simple example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Common HTML Elements

Some common HTML elements you'll use include:

  • Headings: <h1> through <h6>
  • Paragraphs: <p>
  • Links: <a>
  • Images: <img>
  • Lists: <ul>, <ol>, and <li>
  • Divisions: <div>
  • Spans: <span>
  • Forms: <form>, <input>, <button>

CSS Basics

Adding Styles to HTML

There are three ways to add CSS to your HTML documents:

  1. Inline styles: Using the style attribute on HTML elements
  2. Internal stylesheets: Using a <style> tag in the document head
  3. External stylesheets: Linking to a separate CSS file using <link>

CSS Selectors

CSS uses selectors to target HTML elements. Some common selectors include:

  • Element selectors: h1, p, div
  • Class selectors: .classname
  • ID selectors: #idname
  • Descendant selectors: div p
  • Child selectors: div > p

Common CSS Properties

Some common CSS properties you'll use include:

  • color: Text color
  • background-color: Background color
  • font-family: Font type
  • font-size: Font size
  • margin: Space outside an element
  • padding: Space inside an element
  • border: Border around an element
  • display: How an element is displayed (block, inline, flex, etc.)

JavaScript Basics

Adding JavaScript to HTML

There are two main ways to add JavaScript to your HTML documents:

  1. Internal scripts: Using a <script> tag in the document
  2. External scripts: Linking to a separate JavaScript file using <script src="...">

DOM Manipulation

The Document Object Model (DOM) is a programming interface for web documents. JavaScript can interact with the DOM to:

  • Select elements
  • Modify content
  • Change styles
  • Add or remove elements
  • Respond to events

Events

JavaScript can respond to user actions (events) such as:

  • click: When an element is clicked
  • submit: When a form is submitted
  • keypress: When a key is pressed
  • load: When the page finishes loading

AJAX and Fetch API

JavaScript can make HTTP requests to your backend API using:

  • XMLHttpRequest: The original way to make asynchronous requests
  • Fetch API: A modern, promise-based API for making HTTP requests

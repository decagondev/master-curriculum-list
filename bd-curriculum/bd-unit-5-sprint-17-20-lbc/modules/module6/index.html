Module 6: HTML, CSS, and JavaScript

Learning Objectives

  • Understand the fundamentals of HTML for structuring web pages
  • Apply CSS to style and layout web pages
  • Create interactive web experiences using JavaScript
  • Make HTTP requests to your API using Axios

Introduction to Web Development

In this module, you'll learn the fundamentals of web development to create the frontend for your Learn and Be Curious project. A well-designed frontend allows users to interact with your application in an intuitive and engaging way.

Web development uses three core technologies, each with a specific purpose:

  • HTML: Provides the structure and content of web pages
  • CSS: Controls the styling and layout of web pages
  • JavaScript: Adds interactivity and dynamic functionality

Together, these technologies form the foundation of modern web development. By the end of this module, you'll understand how they work together and be able to create a functional frontend for your project.

HTML: Building the Structure

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and content of web pages using a series of elements represented by tags.

Key HTML Concepts

  • Elements and Tags: HTML uses tags like <h1>, <p>, <div> to define elements
  • Attributes: Additional information about elements (e.g., class, id, src)
  • Document Structure: The basic structure includes <html>, <head>, and <body> tags
  • Semantic Elements: Tags like <header>, <nav>, <section> that describe their meaning
  • Links and Images: How to include references to other pages and visual content
  • Forms: Collecting user input with <form>, <input>, <button>, etc.

CSS: Adding Style

What is CSS?

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. CSS controls the visual aspects of web pages, from colors and fonts to layout and animations.

Key CSS Concepts

  • Selectors: How to target HTML elements (element, class, ID, attribute selectors)
  • Properties and Values: Setting visual characteristics like color, font-size, margin
  • The Box Model: Understanding content, padding, border, and margin
  • Layout: Using Flexbox and Grid for positioning elements
  • Responsive Design: Making websites work on various screen sizes with media queries
  • CSS Variables: Defining reusable values for consistent styling

JavaScript: Adding Interactivity

What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It's an essential part of web applications, allowing you to update content dynamically, control multimedia, animate images, and much more.

Key JavaScript Concepts

  • Variables and Data Types: Storing and manipulating information
  • Functions: Reusable blocks of code
  • DOM Manipulation: Interacting with HTML elements
  • Events: Responding to user actions like clicks and form submissions
  • Conditional Logic: Making decisions with if/else statements
  • Loops: Repeating actions with for and while loops
  • AJAX and Fetch: Making asynchronous requests to servers

Making HTTP Requests with Axios

What is Axios?

Axios is a popular JavaScript library for making HTTP requests. It provides an easy-to-use API for performing AJAX operations and works in both browser and Node.js environments.

Key Axios Concepts

  • Setting Up Axios: Including the library in your project
  • Making GET Requests: Retrieving data from an API
  • Making POST Requests: Sending data to an API
  • Handling Responses: Working with the data returned from an API
  • Error Handling: Managing unsuccessful requests
  • Request Configuration: Setting headers, timeouts, and other options

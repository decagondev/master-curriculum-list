Code Implementation

Iterative Depth-First Search

public class Graph {
    private Map> adjacencyList = new HashMap<>();
    
    public void dfsIterative(T start) {
        if (!adjacencyList.containsKey(start)) {
            return;
        }
        
        Set visited = new HashSet<>();
        Stack stack = new Stack<>();
        stack.push(start);
        
        while (!stack.isEmpty()) {
            T vertex = stack.pop();
            if (!visited.contains(vertex)) {
                visited.add(vertex);
                System.out.println(vertex); // Process vertex
                
                // Add unvisited neighbors to stack
                for (T neighbor : adjacencyList.get(vertex)) {
                    if (!visited.contains(neighbor)) {
                        stack.push(neighbor);
                    }
                }
            }
        }
    }

Implementation features:

  • Generic type support for vertices
  • Stack-based iterative approach
  • Efficient visited set tracking
  • Handles cyclic graphs

Breadth-First Search

public void bfs(T start) {
    if (!adjacencyList.containsKey(start)) {
        return;
    }
    
    Set visited = new HashSet<>();
    Queue queue = new LinkedList<>();
    queue.offer(start);
    visited.add(start);
    
    while (!queue.isEmpty()) {
        T vertex = queue.poll();
        System.out.println(vertex); // Process vertex
        
        // Add unvisited neighbors to queue
        for (T neighbor : adjacencyList.get(vertex)) {
            if (!visited.contains(neighbor)) {
                visited.add(neighbor);
                queue.offer(neighbor);
            }
        }
    }
}

Implementation features:

  • Queue-based level traversal
  • Early marking of visited nodes
  • Optimal for shortest paths
  • Level-order processing

Finding Connected Components

public List> findConnectedComponents() {
    Set visited = new HashSet<>();
    List> components = new ArrayList<>();
    
    for (T vertex : adjacencyList.keySet()) {
        if (!visited.contains(vertex)) {
            Set component = new HashSet<>();
            dfsComponent(vertex, visited, component);
            components.add(component);
        }
    }
    return components;
}

private void dfsComponent(T vertex, Set visited, Set component) {
    visited.add(vertex);
    component.add(vertex);
    
    for (T neighbor : adjacencyList.get(vertex)) {
        if (!visited.contains(neighbor)) {
            dfsComponent(neighbor, visited, component);
        }
    }
}

Implementation features:

  • Identifies all connected subgraphs
  • Uses DFS for component discovery
  • Handles disconnected graphs
  • Returns components as sets

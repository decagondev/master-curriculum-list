Implementation Examples

Stack Implementation

public class Stack {
    private ArrayList items;
    
    public Stack() {
        items = new ArrayList<>();
    }
    
    public void push(T item) {
        items.add(item);
    }
    
    public T pop() {
        if (isEmpty()) {
            throw new EmptyStackException();
        }
        return items.remove(items.size() - 1);
    }
    
    public T peek() {
        if (isEmpty()) {
            throw new EmptyStackException();
        }
        return items.get(items.size() - 1);
    }
    
    public boolean isEmpty() {
        return items.isEmpty();
    }
    
    public int size() {
        return items.size();
    }
}

Queue Implementation

public class Queue {
    private LinkedList items;
    
    public Queue() {
        items = new LinkedList<>();
    }
    
    public void enqueue(T item) {
        items.addLast(item);
    }
    
    public T dequeue() {
        if (isEmpty()) {
            throw new NoSuchElementException();
        }
        return items.removeFirst();
    }
    
    public T peek() {
        if (isEmpty()) {
            throw new NoSuchElementException();
        }
        return items.getFirst();
    }
    
    public boolean isEmpty() {
        return items.isEmpty();
    }
    
    public int size() {
        return items.size();
    }
}

Study Tips

To master stacks and queues:

  • Understand LIFO (Stack) and FIFO (Queue) principles thoroughly
  • Practice implementing both data structures from scratch
  • Focus on time complexity of operations
  • Learn to recognize problems that can be solved using stacks or queues
  • Study the various applications in real-world scenarios

Common patterns to look for:

  • Parentheses matching and validation
  • Expression evaluation
  • Monotonic stack problems
  • Sliding window problems using queues

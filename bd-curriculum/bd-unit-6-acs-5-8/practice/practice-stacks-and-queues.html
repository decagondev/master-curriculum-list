Stack Practice Problems

1. Expression Evaluator

Implement a stack-based evaluator for arithmetic expressions:

Requirements:

  • Support operators: +, -, *, /
  • Handle parentheses
  • Process postfix notation
  • Validate input expressions

Example:

ExpressionEvaluator evaluator = new ExpressionEvaluator();
String expr = "2 3 + 4 *";  // (2 + 3) * 4
assertEquals(20, evaluator.evaluate(expr));

2. Browser History

Implement a browser history system using two stacks:

Requirements:

  • Support forward/backward navigation
  • Store page URLs and titles
  • Handle new page visits
  • Clear history functionality

Example:

BrowserHistory history = new BrowserHistory();
history.visit("google.com");
history.visit("leetcode.com");
assertEquals("google.com", history.back());
assertEquals("leetcode.com", history.forward());

Queue Practice Problems

3. Print Queue Manager

Implement a priority print queue system:

Requirements:

  • Support priority levels (1-5)
  • Handle document metadata
  • Implement job cancellation
  • Track print status

Example:

PrintQueue queue = new PrintQueue();
queue.addJob(new PrintJob("Report.pdf", 3));
queue.addJob(new PrintJob("Urgent.doc", 1));
assertEquals("Urgent.doc", queue.processNextJob().getFileName());

4. Message Broker

Implement a simple message broker system:

Requirements:

  • Support multiple topics
  • Handle message persistence
  • Implement subscriber pattern
  • Ensure thread safety

Example:

MessageBroker broker = new MessageBroker();
broker.createTopic("notifications");
broker.subscribe("notifications", new MessageHandler());
broker.publish("notifications", new Message("Hello"));
